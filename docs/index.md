# Table Enhancer for GitHub

Table Enhancer for GitHub is a small, unofficial Chrome extension that makes wide tables easier to read and review in GitHub Markdown file previews.

[Install from the Chrome Web Store](https://chromewebstore.google.com/detail/gnnboemmkojioleagajnphgfdpdemjlg){: .button }
[Read the user guide]({{ site.baseurl }}/user-guide/){: .button }

![A GitHub Markdown table with Table Enhancer controls above it.]({{ site.baseurl }}/store-assets/screenshots/user-guide-overview.png)

## What it helps with

- Keeps wide Markdown tables readable with a dedicated horizontal scroll area.
- Freezes the first displayed rows or left columns while you review large tables.
- Filters, sorts, hides, resizes, fits, wraps, and resets table views without changing the source Markdown.
- Copies the currently visible table view as Markdown, CSV, or TSV.
- Opens a single table in Focus mode when you need more room.

## Where it works

The extension runs only on GitHub Markdown blob file preview pages, such as:

```text
https://github.com/owner/repository/blob/main/docs/example.md
```

It does not run on issues, pull requests, discussions, repository home pages, or Markdown rendered outside GitHub blob file views.

## Privacy at a glance

Table Enhancer for GitHub does not collect, transmit, sell, or share user data. Freeze defaults are saved locally in your browser with Chrome storage.
